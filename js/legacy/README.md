# Legacy modules

В этой папке находятся неиспользуемые или устаревшие модули, оставленные для справки.

Эти файлы **не подключаются** текущим приложением и не участвуют в загрузке данных.
Если потребуется восстановить логику, используйте их только как исторический контекст.

## legacy/state

Файлы в `legacy/state` раньше использовались для работы с локальным состоянием,
но сейчас не участвуют в рантайме.

## Runtime-unused modules

Папки `legacy/ui` и `legacy/calendar` перенесены как неиспользуемые текущим рантаймом.
Они сохранены для истории и возможного возврата.

Дата/причина: cleanup to reduce repo noise.
